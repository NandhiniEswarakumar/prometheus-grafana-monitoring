# Prometheus-Grafana Monitoring System

## Objective
To integrate Prometheus with Grafana for real-time monitoring and visualization.

## Tools Used
- Docker
- Prometheus
- Grafana

## Steps to Run

1. Install Docker
2. Run:
   docker-compose up -d

3. Open:
   - Grafana: http://localhost:3000
   - Prometheus: http://localhost:9090

## Features
- Real-time metrics visualization
- Custom dashboards in Grafana
- Integration with Prometheus data source

## Conclusion
This project successfully integrates Prometheus with Grafana to monitor and visualize real-time system metrics.

## Deliverables
- Grafana Dashboard (dashboard.json)
- Screenshots of output
